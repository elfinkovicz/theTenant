# X (Twitter) Crossposting - Kurzanleitung

## Voraussetzungen
- X Account muss **mindestens 30 Tage alt** sein
- Verifizierte Telefonnummer und E-Mail
- X Premium ist **nicht** erforderlich

## Einrichtung (8 Schritte)

1. **Developer Portal öffnen** → [developer.x.com](https://developer.x.com) und anmelden

2. **Projekt erstellen** → "Create Project" → Namen vergeben

3. **App erstellen** → Im Projekt eine neue App anlegen

4. **Permissions setzen** → "User authentication settings" → "Read and write" aktivieren

5. **Callback URL eintragen** → `https://viraltenant.com` (oder deine Custom Domain)

6. **Keys generieren** → Unter "Keys and Tokens":
   - API Key (Consumer Key)
   - API Secret (Consumer Secret)
   - Access Token
   - Access Token Secret

7. **In ViralTenant einfügen** → Crossposting → X (Twitter) → Alle 4 Keys eintragen

8. **Verbindung testen** → Button "Verbindung testen" klicken

## Limits (Free Tier)
- 1.500 Posts/Monat (ca. 50/Tag)
- Text, Bilder und Videos unterstützt
